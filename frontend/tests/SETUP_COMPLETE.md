## 🐛 Troubleshooting

### Tests Fail with "Connection Refused"

**Problem**: Dev server không chạy  
**Solution**: Start server trước: `npm run dev`

### Browser Not Found

**Problem**: Playwright browsers chưa được cài  
**Solution**: `npx playwright install chromium`

### Port Already in Use

**Problem**: Port 3000 đang được sử dụng  
**Solution**: 
```bash
lsof -ti:3000 | xargs kill -9
```
